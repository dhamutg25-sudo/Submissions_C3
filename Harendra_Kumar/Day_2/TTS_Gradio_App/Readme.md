# Text-to-Speech with gTTS

A simple Python project that converts text to speech using **gTTS (Google Text-to-Speech)**.  
Supports multiple languages and can generate audio files from text input. Optionally includes a **Gradio web interface** for interactive usage.

---

## Features

- Convert text to speech in multiple languages.
- Save speech output as **MP3** files.
- Optionally use **slow mode** for a slower voice.
- Gradio interface for interactive text-to-speech conversion.
- Works offline after installation (requires internet for gTTS).

---

## Installation

- pip install gtts
- pip install gradio
